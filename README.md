# Barcode Generator & ZPL Converter

Generate sequential barcodes in Excel and convert them to ZPL format for Zebra printers.

## Files

- `barcode_generator.py` - Generates sequential barcodes to Excel
- `zpl.py` - Converts Excel barcodes to ZPL format

## Usage

```bash
cd "/home/dataopske/Desktop/FINDSTD BARCODES"
```

Generate barcodes:
```bash
python3 barcode_generator.py
```

Convert to ZPL:
```bash
python3 zpl.py
```

## Requirements

```bash
pip install openpyxl
```
