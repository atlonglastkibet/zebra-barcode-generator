# Barcode Generator & ZPL Converter

Generate sequential barcodes in Excel and convert them to ZPL format for Zebra printers.

## Files

- [barcode_generator.py](/home/dataopske/Desktop/FINDSTD BARCODES/barcode_generator.py) - Generates sequential barcodes to Excel
- [zpl.py](/home/dataopske/Desktop/FINDSTD BARCODES/zpl.py) - Converts Excel barcodes to ZPL format

## Usage

Generate barcodes:
```bash
python3 "/home/dataopske/Desktop/FINDSTD BARCODES/barcode_generator.py"
```

Convert to ZPL:
```bash
python3 "/home/dataopske/Desktop/FINDSTD BARCODES/zpl.py"
```

## Requirements

```bash
pip install openpyxl
```
